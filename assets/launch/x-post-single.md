## Copy-paste text

I built **Grok Executive Suite** — a native macOS GUI for **Grok Build**.

I love Grok, but I prefer delegating through a GUI over living in Terminal. This wraps Grok Build's full agentic power in one desktop workspace built around one idea:

**Reduce everything on your computer to executive functions.** You decide *what* needs done. Grok handles *how*.

**What it does:**
• Persistent projects & resumable task threads
• Scheduled cron jobs with isolated run history
• Working-folder context injected into every prompt
• Per-task progress checklists & connector tracking
• Act-without-asking autonomy toggle
• Desktop capture, message queue, spellcheck

It's Grok Build — with an executive desk.

**Install (Apple Silicon Mac):**
1. Grok Build CLI (SuperGrok / Premium+):
   `curl -fsSL https://x.ai/cli/install.sh | bash`
2. Download v0.3.3 DMG from GitHub Releases
3. Drag to Applications
4. First open: right-click → Open (unsigned community build)
5. + New Project → pick a folder → start delegating

**Important:** This is a **community project**, not an official xAI product. I'm an enthusiastic Grok user — not an xAI employee — who prefers GUIs and wanted to contribute something useful to the Grok community.

Huge thanks to **Elon** and the **xAI team** for Grok Build and the continued support they give builders, engineers, and entrepreneurs pushing innovation forward.
